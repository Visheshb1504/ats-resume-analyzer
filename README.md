# ATS Resume Analyzer

This project evaluates resumes using keyword-based analysis to simulate an Applicant Tracking System (ATS).

## Overview

* Approach: Keyword matching
* Input: Resume text
* Output: ATS score and matched keywords

## Features

* Loads predefined job-related keywords
* Calculates match percentage
* Displays missing and matched keywords

## Tech Stack

* Python

## How to Run

1. Place your resume text in `samples/sample_resume.txt`
2. Run:

   ```
   python app/app.py
   ```
3. View ATS score and matched keywords

## Use Case

Helps candidates optimize resumes based on job-specific keywords.
